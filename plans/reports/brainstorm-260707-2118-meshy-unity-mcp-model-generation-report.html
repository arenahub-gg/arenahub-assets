■ 02 · Problem 02 / 08

Problem-first inversion

Đề xuất là MCP.
Vấn đề là năng lực tạo asset.

Repo chỉ tiêu thụ asset có sẵn — 207 pack Kenney CC0. Khi game cần sprite hay model đặc thù không có trong thư viện, pipeline bí hoàn toàn.

Ba vấn đề gốc được giải nén từ đề xuất "thêm MCP cho Meshy và Unity":

Một — không có đường tạo asset mới theo yêu cầu. Hai — 48 pack 3D Kenney đang "chết" vì engine chỉ render Canvas 2D. Ba — năng lực generate phải dùng chung cho nhiều người clone repo, không phải setup cá nhân một máy.

Unity không sinh model — nó là editor tiêu thụ model. Đưa Unity vào cùng scope với Meshy là trộn hai bài toán khác nhau.

■ 03 · Evidence 03 / 08

Codebase facts & external verification

Những gì đã kiểm chứng.

01

Stack

Canvas 2D thuần

TypeScript strict + Vite, zero engine framework. Không .mcp.json, không three.js, không Unity. Games build ra static dist.

02

Pipeline

Multi-source sẵn sàng

Asset đổ vào assets/{source}-{slug}/, resolver tách riêng, license theo dõi per-pack với cảnh báo CC-BY attribution. Tiền lệ commit binary: 2 pack R1.

03

Dead weight

48

pack 3D Kenney (GLB/GLTF) nằm trong manifest mà không game nào dùng được — thiếu runtime 3D.

04

Meshy — verified

MCP chính thức tồn tại

@meshy-ai/meshy-mcp-server trên npm. Tools: text-to-3d, image-to-3d, refine, remesh, retexture, rig, animate. Cần MESHY_API_KEY. Free plan → output CC BY 4.0; paid → user sở hữu.

05

Unity — verified

MCP chính thức nhưng machine-local

Đi kèm AI Assistant package: Unity 6+, subscription, link Unity Cloud, Editor phải đang chạy. Không thể project-scope cho nhiều user qua repo config.

■ 05 · Recommendation 05 / 08

Decision

Phương án A, bốn phase.
Unity tách khỏi đường generate.

Phase 1 — Core (làm ngay)

1. .mcp.json root repo khai báo server Meshy chính thức qua npx, API key từ env mỗi user. 2. Mở rộng schema pack.json cho asset generated: generator, provenance (prompt, taskId, plan), license hai nhánh CC-BY-4.0 / proprietary-owned. 3. Skill .claude/skills/meshy-model-gen/: generate → download GLB → ingest assets/meshy-{slug}/ → catalog; luật cứng: pack CC-BY chỉ vào game có attribution surface. 4. Binary generated commit thẳng git (duy nhất, không tái tải được). 5. Docs + cập nhật CLAUDE.md.

P2

Phase 2

Sprite-render pipeline

GLB → sprite sheet turntable cho engine Canvas 2D hiện tại. Cần spike headless GL trên Windows trước khi cam kết (fallback: Blender CLI).

P3

Phase 3

Template three.js

templates/threejs-game/ + một game 3D tham chiếu, theo đúng extraction-over-speculation. Unlock 48 pack 3D + model Meshy.

P4

Phase 4 — Unity

Docs-only, brainstorm riêng

Hướng dẫn setup user-level (Unity 6+, subscription, Unity Cloud). Game Unity WebGL phá nguyên tắc vendoring/static-dist — cần thiết kế lại riêng khi kích hoạt.
